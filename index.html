Cellcounter

Cell counter is an application to manually count cells and can be use in hematology or somewhere else.

Cellscounter ?

CellsCounter is a cute application to count cells. I made it for my own purpose at my work. Indeed, it can be use in haematology but anywhere else. You can open different templates and count what you want. From you keyword or with your mouse, increase and decrease each elements until you reach the limit. At the end, you could check the perents of each items.

Installation

You can download the binary for windows , or get the source code and compile it for Linux or Mac. This appliation depends on Qt 5.3. So you can easily compile it by downloading Qt and open the cells.pro file from QtCreator
